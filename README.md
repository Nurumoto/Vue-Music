应用名称：小雨滴音乐播放器（基于qq音乐api）
作者：潇禹
使用框架Vue2.0
使用依赖：animate.css、vue-router、vue-resource、webpack、vue-awesome-swiper...
项目结构：
   
   |- facicon.ico
   |- index.html
   |- package.json
   |- webpack.config.js
   |- src
   |   |- assets
   |   |- component
   |   |- json
   |   |- store
   |   |- App.vue
   |   |- main.js
   |   |- router.config.js

已实现功能：

   音乐播放
   自动播放下一曲
   音乐列表（增删）
   下一曲
   排行榜
   排行榜详情页
   搜索
   loading
   播放详情页
   滚动歌词
   展示音乐进度
